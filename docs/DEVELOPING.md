# Developing

## Prerequisites

- Node.js >= 20: https://nodejs.org/

- Rust: https://rustup.rs/

- PNPM: https://pnpm.io/

- Tauri: https://tauri.app/guides/prerequisites/

## Build from source

Clone project

```
git clone https://github.com/lumehq/lume.git && cd lume
```

Install required dependencies

```
pnpm install
```

Run dev

```
pnpm tauri dev
```

Build

```
pnpm tauri build
```
